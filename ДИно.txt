Динозавр: 
	dino_x - статичная, в лево и в право передвижения нет
	dino_y - отвечает за "стоит на месте или прыгнул"
препятствие наземное:
	kaktus_x - передвигается от правого края к левого
	kaktus_y - статичное, выше пола
	массив кактусов [тройные кактусы, большой кактус, маленький кактус.]
	массив ям
препятствие воздушное:
	bird_x - передвигается от правого края к левого
	bird_y - статичное, ниже потолка
	массив птиц [большая птица, маленькая птица]

счетчик:
	count - +1 за пройденное препятствие, нужен для увеличения скорости/cложности


dino_y (Регистр, 8 бит): Хранит текущую Y-координату динозавра.
dino_velocity (Регистр, 8 бит, знаковый): Хранит вертикальную скорость динозавра
dino_duck_down (Регист, 1 бит): хранит состояние пригнулся или нет
kaktus_x (Регистр, 8 бит): Хранит текущую X-координату препятствия.
game_tick: Определять тикрейт - чтобы игра шла с нормальной скоростью
делитель частоты - начальное значение d, если count увеличился на 5 - d *= 1.1
count: счетчик игровой
health: жизни 3

dino_x: C
kaktus_y: C
ground - C
bird_y: C
left_bridge: левая границы в пикселях - С
right_bridge: правая границы в пикселях - С
ширина каждого объекта:
dino_wheight: C
bird_wheight: C
kaktus_wheight: C
gameover (Триггер, 1бит): 




Логика взаимодействия
Соприкосновение (коллизия), когда динозавр пересекается с препятствия.
Как только пересечение - остановка игры, счетчик на экран

Действия в игре
Обновление позиции каждый тик, препятствие движется к левому краю, как только касается края - Сount +1
Реализация прыжка - скорость прыжка - если прыгнул, то флаг 1, чтобы второй раз не прыгал - проверка на приземлелся
Прыжок по кнопке
1. если y < 0 - провалился в яму, частичная инициализация, -health
2. если y >= 0 и координата x совпадает с кактусом или с птицей - частичная инициализация, -health
3. если health == 0 => game over
4. частичная инициализация - очистка ленты, если яма, координата y = 0;
5. если dino_duck_down == 1 и меняем макет (кнопка опущена вернуть старый макет)
6. если dino_duck_down == 1, то он не может прыгнуть
7. если y > 0, то dino_duck_down остается 0
8. game over: обнуление счетчика, очистка экрана, очистка регистров, картинка "GAME OVER", делитель частоты = d
9. заставка: счетчик = 0, делитель частоты = d, экран пуст
10. если count mod 100, замена фона
11. физика прыжка: y_max = v0^2/2g - поднимается пока y < y_max, если y == y_max, то падаем, пока y != 0 (v0 = const), вверх = (v = v0-gt), вниз = (v = gt)
12. если падает в яму: падает, пока координаты ног не совпадут с дном ямы. (v = gt)
13. кактусы и ямы двигаются с одной скоростью
14. птицы двигаются с большей скоростью, большая птица еще быстрее
15. при столкновении учитывать только координаты всех крайних точек
